# Matplotlib

Matplotlib es una biblioteca de visualización que permite crear gráficos de alta calidad de manera similar a MATLAB. Normalmente se importa como `plt`:

```python
import matplotlib.pyplot as plt
```

## Gráfico de líneas

La función `plt.plot(x, y)` dibuja una línea que conecta los puntos. Se pueden agregar múltiples líneas llamando varias veces a `plot` o pasando varios pares de datos.

```python
x = [1, 2, 3, 4, 5, 6, 7, 8]
y1 = [2, 4, 6, 8, 10, 12, 14, 16]
y2 = [1, 4, 9, 16, 25, 36, 49, 64]

plt.plot(x, y1, "-or")   # línea sólida, círculos rojos
plt.plot(x, y2, "--*m")  # línea discontinua, estrellas magenta
plt.show()
```

## Estilos de línea, color y marcadores

En `plot` se puede usar una cadena de formato de tres partes: `[línea][marcador][color]`. Por ejemplo:

- `'-or'`:  línea sólida (`-`), círculos (`o`), rojo (`r`)
- `'--*m'`: línea discontinua (`--`), estrellas (`*`), magenta (`m`)
- `':g'`: línea punteada (`:`), verde (`g`)
- `'^b'`: triángulos (`^`), azul (`b`), sin línea

## Etiquetas, título y leyenda

```python
plt.title("Gráfico con varias líneas")
plt.xlabel("Números")
plt.ylabel("Resultados")
plt.legend(["*2", "^2"])   # Orden según las llamadas a plot
```

La leyenda asocia automáticamente las etiquetas con cada trazo.

## Gráfico de dispersión (`scatter`)

Para puntos individuales sin conexión se usa `plt.scatter`.

```python
x = np.random.randint(1, 100, 10)
y = np.random.randint(1, 100, 10)

plt.scatter(x, y, facecolors="none", edgecolors="b")  # círculos vacíos con borde azul
plt.show()
```

Parámetros comunes:
- `facecolors` o `facecolor`: color de relleno (`"none"` para transparente)
- `edgecolors` o `edgecolor`: color del borde
- `s`: tamaño de los marcadores
- `alpha`: transparencia

## Visualización de matrices con `imshow`

Cuando se trabaja con matrices, `plt.imshow` muestra la matriz como una imagen en color.

```python
plt.imshow(matriz, cmap="hot")
plt.title("Tiempo = 0 s")
plt.show()
```

- `cmap`: mapa de color predefinido (`"hot"`, `"viridis"`, `"plasma"`, `"gray"`, etc.)
- `vmin`, `vmax`: fijan el rango de valores para la escala de colores

## Mapas de color personalizados (`ListedColormap`)

Para representar categorías se crea un mapa de color con colores específicos.

```python
from matplotlib.colors import ListedColormap

colores = ListedColormap(["black", "green", "red", "gray"])

plt.imshow(matriz, cmap=colores, vmin=0, vmax=4)
```

- Los colores se asignan en orden: el valor 0 primer color (`"black"`), valor 1 segundo (`"green"`), etc.
